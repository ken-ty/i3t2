### 基底クラス

| 基底クラス名 | 役割 |
| --- | --- |
| `PageBase` | 全ての `page` クラスが継承する. | 
| `ApiBase` | 全ての `API` クラスが継承する. | 
| `ModelBase` | 全ての `Model` クラスが継承する. | 
| `ControllerBase` | 全ての `Controller` クラスが継承する. |
